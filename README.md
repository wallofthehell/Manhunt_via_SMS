# 어딨냐 (Manhunt via SMS)

이 프로젝트는 통신영장 등을 통해 확보한 기지국 접속 로그 문자메시지(CSV 등)를 분석하여, 지도상에 대상자의 이동 동선을 시각화해 주는 안드로이드 데이터 분석 도구입니다.

## 주요 기능
- **주소 추출 및 변환**: 정형/비정형 주소 문자열에서 위치 정보를 추출하고 좌표로 변환합니다.
- **동선 시각화**: OpenStreetMap을 기반으로 점과 점 사이를 이은 직선뿐만 아니라, 도로망을 따라 이동하는 실제 동선 경로를 시각화합니다.
- **체류 시간 분석**: 동일한 위치에 연속해서 나타나는 기록을 병합하여 해당 장소에 머무른 시간을 자동으로 산출합니다.
- **데이터 내보내기**: 분석 및 정형화된 주소 목록을 다시 CSV 형식으로 기기에 다운로드할 수 있습니다.

## 사용 방법
1. 통신사로부터 받은 SMS 로그를 준비합니다.
2. 앱을 실행하고 로그를 분석합니다.
3. [앱 내부 동선 지도 보기] 기능이 제거되었으므로, 추출된 리스트를 터치하여 구글 지도로 개별 확인하거나 CSV로 내보내어 외부 도구에서 확인합니다.
